Programmer's Klondike - электронная библиотека программиста. Здесь Вы
сможете огромное количество книг разной тематики совершенно бесплатно.
На сайте имеются следующие разделы:

Assembler - Книги по программированию на Ассемблере.

C/C++ - Книги по C, C++, стандартной библиотеке шаблонов STL и другие.

Computer Science - Книги по всем разделам вычислительных наук (алгоритмы, паттерны, кодирование информации).

Cryptography - Книги по криптографии и криптоанализу

Database - Книги по Базам Данных (Основы реляционных баз данных, SQL, MySQL, Oracle и другие)

Development - Книги-помощники по разработке программного обеспечения на всех стадиях развития.

.NET - Книги, связанные с новейшей и многообещающей платформой Microsoft .NET

Hacking - Книги, в которых описаны процессы взлома, что поможет в создании эффективных мер по обеспечению безопасности компьютерных систем.

Java - Книги по Java (Java 2 Enterprise Edition,Java 2 Micro Edition для встраиваемых устройств)

OOA&D - Книги по Объектно-Ориентированному Программированию, Анализу и Дизайну (Проектированию).

HTML - Книги по языку гипертекстовой разметки документов - HTML.

Perl - Книги по программированию на языке Perl.

PHP - Книги по PHP-программированию.

JavaScript - Книги по созданию Web-сайтов с помощью JavaScript.

Web-Design - Книги, которые помогут вам создать Web-сайт или улучшить его внешний и внутренний мир.

UML - Книги по Унифицированному Языку Объектно-Ориентированного Моделирования (UML).

Usability - Книги, дающие рекомендации по построению интерфейсов программ, а также дизайну как таковому.

*nux/FreeBSD - Книги по Unix-подобным операционным системам: Unix, Linux, FreeBSD...

Other - Книги, которые не подошли по тематике к имеющимся разделам.

Будем рады видеть Вас на нашем сайте.
